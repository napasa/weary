过程蛮长的，大概如下所述。从网上了解到想学Qt语言方面就得先学C++，平台方面又得知Qt是开源的在linux下能更好
的深入了解他，因此又先得有unix系的操作系统经验。我便按照这个思路来， 先后看了<< C++ Primer >>,
<<鸟哥的linux私房菜>>，<< unix网络编 程>>， << unix高级编程 >>, << 计算机网络 >>， << shell高级编程 >>，
